## 原始内容

**团队甲**：便宜的 judge 足够用，成本收益比更高。  
**团队乙**：高价 judge 更稳定，尤其在边界题上更可靠。  
**团队甲**：预算有限时不应该默认上高价模型。  
**团队乙**：如果追求稳定性，便宜模型的波动成本反而更高。

双方都提供了理由，但没有达成统一结论。
