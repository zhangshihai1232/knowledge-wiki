## 原始内容

长期记忆写入只允许记录高价值、可复用、非敏感事实。

- 低价值或一次性信息不应写入长期记忆。
- 敏感内容不得进入长期记忆，即使它看起来“以后可能有用”。
- 这是一组约束条件，不是鼓励性总结。
